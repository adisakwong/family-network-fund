# Family Network Fund 👨‍👩‍👧‍👦💸

แอปพลิเคชันเว็บแอป (Web App) สำหรับจัดการข้อมูลสมาชิกและระบบกองทุนเครือญาติ ใช้งานง่ายผ่านสมาร์ทโฟนและคอมพิวเตอร์ โดยมีระบบเบื้องหลัง (Backend) ทำงานบน **Google Apps Script (GAS)** และใช้ **Google Sheets** เป็นฐานข้อมูลหลัก 

---

## ✨ ฟีเจอร์หลัก (Key Features)

### 👥 ระบบจัดการสมาชิก (Member Management)
- **ลงทะเบียนสมาชิกใหม่:** กรอกข้อมูลส่วนตัว, ที่อยู่, เบอร์โทร, ชื่อเล่น พร้อมแนบรูปถ่ายโปรไฟล์ได้ทันที
- **อ่านข้อบังคับก่อนสมัคร:** มีปุ่มกดเพื่ออ่าน **"ข้อบังคับกองทุน"** (regulations.pdf) ในหน้าลงทะเบียน
- **แสดงโพรไฟล์สมาชิก:** ค้นหาและดูข้อมูลสมาชิกทั้งหมด รวมถึงความสัมพันธ์กับสมาชิกหลัก (ครอบครัว/ญาติ)
- **ระบบถ่ายภาพในตัว (In-App Camera):** ถ่ายรูปเซลฟี่หรืออัปโหลดรูปภาพผ่านเบราว์เซอร์ได้ทันที 
- **ระบบจัดเก็บรูปภาพอัตโนมัติ:** รูปภาพจะถูกส่งไปเก็บไว้ใน Google Drive อัตโนมัติ

### 💰 ระบบรายงานการเงิน (Financial Status)
- **หน้าสถานะทางการเงิน:** ดึงข้อมูลรายรับ-รายจ่ายจาก Google Sheets (แผ่นงาน `financial`) มาแสดงผลแยกรายเดือน
- **คำนวณยอดคงเหลือ:** ระบบสรุปยอดรวม "รายรับ" - "รายจ่าย" และ "ยอดเงินคงเหลือสุทธิ" ให้ทันที
- **ตารางที่อ่านง่าย:** ออกแบบให้ Responsive แสดงผลได้ดีทั้งบนคอมพิวเตอร์และมือถือ

### 💸 ระบบแจ้งผลการส่งเงิน (Payment Reporting)
- **แจ้งโอนเงินออนไลน์:** สมาชิกสามารถส่งหลักฐานการโอนเงิน (สลิป) ผ่านหน้าโปรไฟล์ของตนเองได้
- **ระบบผู้ดูแล (Admin Authority):** ผู้ใช้ที่อยู่ในรายชื่อ `admin` (อิงจากรหัสบัตรประชาชน) สามารถเข้าไปแจ้งผลการส่งเงินแทนสมาชิกท่านอื่นได้ (มีประโยชน์สำหรับผู้สูงอายุที่ไม่สะดวกใช้งานเอง)
- **แยกไฟล์จัดเก็บข้อมูลธุรกรรม:** ข้อมูลการโอน (สลิป) จะถูกแยกไปบันทึกเก็บไว้ในไฟล์ Google Sheets อันใหม่ (สร้างอัตโนมัติ) เพื่อป้องกันฐานข้อมูลหลักบวม
- **อัปโหลดสลิปอัตโนมัติ:** รูปภาพสลิปจะถูกส่งไปเก็บไว้ในโฟลเดอร์ `slips` ใน Google Drive

### 🔔 ระบบแจ้งเตือน (Real-time Notifications)
- **Telegram Bot API:** เมื่อมีการแจ้งโอนเงิน หรือบันทึกรายจ่ายต่างๆ ระบบจะส่งข้อความแจ้งเตือนพร้อมรูปภาพสลิปไปยังกลุ่ม Telegram ของคณะกรรมการทันที
- **ระบบสรุปยอดอัตโนมัติ (Automated Summary):** ดึงยอดรวมจากตาราง `transactions` และส่งรายงานสรุป รายรับ-รายจ่าย เข้า Telegram อัตโนมัติในตอนสิ้นวันและสิ้นเดือน (รองรับการตั้งเวลาผ่าน Google Apps Script Time-driven triggers)

## 🛠 เทคโนโลยี (Tech Stack)

- **Frontend:** HTML5, CSS3, JavaScript (ES6+), SweetAlert2
- **Backend:** [Google Apps Script (GAS)](https://developers.google.com/apps-script)
- **Database:** [Google Sheets](https://docs.google.com/spreadsheets/)
- **Storage:** [Google Drive](https://www.google.com/drive/) 
- **Notification:** [Telegram Bot API](https://core.telegram.org/bots/api)

## 🚀 วิธีการติดตั้ง (Quick Start Guide)

### ส่วนที่ 1: การตั้งค่าระบบหลังบ้าน (Google Apps Script)
1. **สร้าง Spreadsheet:** เตรียมไฟล์ Google Sheets เป็นฐานข้อมูล
2. **เปิด Apps Script:** ไปที่เมนู `Extensions` (ส่วนขยาย) > `Apps Script`
3. **วางโค้ด:** คัดลอกโค้ดจากไฟล์ `Code.gs` สคริปต์นี้ไปวางทับในหน้าต่าง
4. **ตั้งค่า Telegram Bot (ระบบแจ้งเตือน):**
   - ขอ Bot Token ผ่าน BotFather ใน Telegram และหาค่า Chat ID ของกลุ่ม
   - แก้ไขตัวแปร `TG_BOT_TOKEN` และ `TG_CHAT_ID` ในไฟล์ `Code.gs`
   - หากต้องการส่งสรุปยอดอัตโนมัติ ให้ไปตั้งค่า Time-driven triggers ใน Apps Script ให้รันฟังก์ชัน `sendDailySummary` และ `sendMonthlySummary`
5. **Deploy เป็น Web App:**
   - กดปุ่ม **Deploy** > **New Deployment** > **Web App**
   - **Execute as:** "Me"
   - **Who has access:** "Anyone"
6. **ขอสิทธิ์การเข้าถึง:** กดยืนยันสิทธิ์อนุญาต

### ส่วนที่ 2: การเชื่อมต่อหน้าบ้าน (Frontend Connection)
1. เปิดไฟล์ `script.js` (สำหรับระบบหลัก) หรือแท็ก `<script>` ในไฟล์ `index.html`, `register.html`, `finance.html`
2. อัปเดตตัวแปรเชื่อมต่อไปยังบอท
   ```javascript
   const GAS_WEB_APP_URL = 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec';
   ```

## 📂 โครงสร้างข้อมูลใน Google Sheets

*รายละเอียดของตารางอ้างอิงและคีย์ต่างๆ (ดูตัวอย่างแผ่นงานและคอลัมน์ได้ที่ Google Sheets ระบบ)*
- `members`: ข้อมูลสมาชิกทั่วไป เช่น รหัสบัตรปชช., ชื่อ, นามสกุล, เบอร์โทร, วันเกิด, รูปภาพ, ชื่อเล่น, ฯลฯ
- `admin`: รายชื่อรหัสบัตรประชาชนของผู้ดูแลระบบ
- `payment`: ข้อมูลการแจ้งโอน/รายจ่าย
- `financial`: สรุปข้อมูลการเงิน (เพื่อไปแสดงบนหน้ารายงานทางการเงิน)

---

## 📸 รายละเอียดทางเทคนิค
- **Leading Zeros:** รองรับการพิมพ์รหัสประชาชนหรือเบอร์โทร `0` นำหน้า โดยข้อมูลไม่เพี้ยน
- **Selfie Mode:** กล้องทำงานได้ดีบนอุปกรณ์มือถือ
- **Zero Cost:** รันบน Google Infrastructure ฟรี 100%
