# 🏥 Family Network Fund (กองทุนเครือญาติเพื่อสุขภาพและพิธีศพ)

[![Platform](https://img.shields.io/badge/Platform-Google%20Apps%20Script-4285F4?logo=google-apps-script&logoColor=white)](https://developers.google.com/apps-script)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![UI](https://img.shields.io/badge/UI-Modern%20Responsive-blueviolet)](https://github.com/features/actions)

**Family Network Fund** เป็นเว็บแอปพลิเคชันแบบ Serverless ที่ออกแบบมาเพื่อบริหารจัดการกองทุนภายในครอบครัวหรือเครือญาติ โดยเน้นความง่ายในการใช้งาน ความปลอดภัยของข้อมูล และความสวยงามทันสมัย รองรับการใช้งานบนมือถือ (Mobile-First) อย่างสมบูรณ์ 100%

---

## ✨ ฟีเจอร์เด่น (Key Features)

### 👤 ระบบจัดการสมาชิก (Member Management)
- **เข้าสู่ระบบปลอดภัย:** ล็อกอินด้วยรหัสประจำตัวประชาชน 13 หลัก
- **ข้อมูลสมาชิกครบถ้วน:** บันทึกทั้งชื่อ-นามสกุล, ชื่อเล่น, วันเปิดบัญชี, วันเกิด, โรคประจำตัว, และข้อมูลผู้รับผลประโยชน์
- **ระบบถ่ายภาพในตัว (In-App Camera):** ถ่ายรูปเซลฟี่หรืออัปโหลดรูปภาพผ่านเบลารว์เซอร์ได้ทันที ไม่ต้องสลับแอป
- **ระบบจัดเก็บรูปภาพอัตโนมัติ:** รูปภาพจะถูกส่งไปเก็บไว้ใน Google Drive ส่วนตัวของคุณโดยอัตโนมัติ พร้อมการตั้งค่าสิทธิ์ให้เข้าชมได้ทันที

### 💰 ระบบรายงานการเงิน (Financial Status)
- **สรุปผลรายเดือน:** ดึงข้อมูลรายรับ-รายจ่ายจาก Google Sheets มาแสดงผลแบบเรียลไทม์
- **คำนวณยอดคงเหลือ:** ระบบสรุปยอดรวมรายรับ รายจ่าย และยอดเงินคงเหลือสุทธิให้ทันที
- **ตารางที่อ่านง่าย:** ออกแบบให้แสดงผลได้ดีทั้งบนหน้าจอคอมพิวเตอร์และมือถือ

### 💸 ระบบแจ้งผลการส่งเงิน (Payment Reporting)
- **แจ้งโอนเงินออนไลน์:** สมาชิกสามารถส่งหลักฐานการโอนเงิน (สลิป) ได้โดยตรงผ่านหน้าโปรไฟล์
- **อัปโหลดสลิปอัตโนมัติ:** รูปภาพสลิปจะถูกส่งไปเก็บไว้ในโฟลเดอร์ `slips` ใน Google Drive
- **แยกไฟล์จัดเก็บ (External Storage):** ข้อมูลการโอนจะถูกแยกไปเก็บไว้ในไฟล์ชื่อ `payment` เพื่อเพิ่มประสิทธิภาพในการทำงานระยะยาว
- **ระบบผู้ดูแล (Admin Authority):** ผู้ใช้ที่อยู่ในรายชื่อ `admin` สามารถแจ้งผลการส่งเงินแทนสมาชิกท่านอื่นได้ทุกคน

### 🎨 ประสบการณ์ผู้ใช้ที่เหนือระดับ (Premium UI/UX)
- **Modern Design:** ใช้โทนสีเข้มพรีเมียม (Dark Blue Gradient) ให้ความรู้สึกน่าเชื่อถือและทันสมัย
- **Responsive Layout:** ปรับเปลี่ยนหน้าตาตามขนาดหน้าจอ (มือถือ/แท็บเล็ต/คอมพิวเตอร์)
- **Micro-Animations:** มีเอฟเฟกต์การตอบสนองที่ลื่นไหล สร้างประสบการณ์ที่ดีในการใช้งาน

## 🛠 เทคโนโลยี (Tech Stack)

- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Backend:** [Google Apps Script (GAS)](https://developers.google.com/apps-script) — ทำงานเป็น Serverless API
- **Database:** [Google Sheets](https://docs.google.com/spreadsheets/) — ใช้จัดเก็บข้อมูลสมาชิกและการเงิน
- **Storage:** [Google Drive](https://www.google.com/drive/) — ใช้จัดเก็บรูปภาพสมาชิก

## 🚀 วิธีการติดตั้ง (Quick Start Guide)

### ส่วนที่ 1: การตั้งค่าระบบหลังบ้าน (Google Apps Script)
1. **สร้าง Spreadsheet:** สร้างไฟล์ Google Sheets ใหม่ (เช่นชื่อ "Family Fund Database")
2. **เปิด Apps Script:** ไปที่เมนู `Extensions` (ส่วนขยาย) > `Apps Script`
3. **วางโค้ด:** คัดลอกโค้ดจากไฟล์ `Code.gs` ในโปรเจกต์นี้ไปวางทับในหน้าต่างสคริปต์
4. **Deploy เป็น Web App:**
   - กดปุ่ม **Deploy** (การทำให้ใช้งานได้) > **New Deployment** (การทำให้ใช้งานได้รายการใหม่)
   - เลือกประเภทเป็น **Web App**
   - ตั้งค่า **Execute as:** "Me" (ตัวฉัน)
   - ตั้งค่า **Who has access:** "Anyone" (ทุกคน) *— จำเป็นเพื่อให้สมาชิกคนอื่นเข้าถึงได้*
5. **ขอสิทธิ์การเข้าถึง:** กดยืนยันสิทธิ์ (Authorize) ให้สคริปต์เข้าถึง Sheets และ Drive ของคุณ
6. **คัดลอก URL:** บันทึกค่า **Web App URL** (ที่ลงท้ายด้วย `/exec`) ไว้สำหรับขั้นตอนถัดไป

### ส่วนที่ 2: การเชื่อมต่อหน้าบ้าน (Frontend Connection)
1. เปิดไฟล์ `script.js` หรือ `index.html` (ตรวจสอบจุดที่มีตัวแปร `GAS_WEB_APP_URL`)
2. นำ URL ที่ได้จากขั้นตอนก่อนหน้ามาวางแทนที่:
   ```javascript
   const GAS_WEB_APP_URL = 'https://script.google.com/macros/s/.../exec';
   ```
3. บันทึกไฟล์ และคุณสามารถเปิด `index.html` บนเบราว์เซอร์เพื่อเริ่มใช้งานได้ทันที!

## 📸 รายละเอียดทางเทคนิคที่สำคัญ
- **Leading Zeros:** ระบบรองรับการพิมพ์รหัสประชาชนหรือเบอร์โทรที่ขึ้นต้นด้วย `0` โดยข้อมูลจะไม่หายเมื่อบันทึกลง Google Sheets
- **Selfie Mode:** กล้องรองรับทั้งกล้องหน้าและกล้องหลังสำหรับอุปกรณ์มือถือ
- **Zero Cost:** ระบบทั้งหมดรันบน Google Infrastructure ฟรี 100% (ภายใต้โควตาของ Google)
